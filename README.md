# Anonymous-Hacker
Anonymous OS Linux with tor

# Setup 

<pre>
$ git clone https://github.com/teeknofil/Anonymous-Hacker.git
$ cd Anonymous-Hacker
$ chmod +x install.sh
$ ./install.sh
</pre>

# How run 

<pre>
$ sudo anonymous-hacker start
</pre>

# How stop

<pre>
$ sudo anonymous-hacker stop
</pre>

# How know status connect

<pre>
$ sudo anonymous-hacker status
</pre>

# Note
setup manualy noip.
